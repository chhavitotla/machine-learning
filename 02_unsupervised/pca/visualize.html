Principal Component Analysis

A correlated cloud has a favorite direction — the axis along which it spreads the most. PCA finds it by eigendecomposing the covariance matrix: PC1 (cyan) points along maximum variance, PC2 (magenta) along what's left. Drag the projection slider and watch every point slide onto the PC1 line — 2D collapsing to 1D while losing as little as mathematically possible.

arrow lengths ∝ √eigenvalue (the std along each component) · faint stems = what projection throws away
Drag rotate data and watch the eigenvectors re-aim instantly — PCA has no preferred axes, it reads direction straight out of the covariance. Set ρ ≈ 0 and PC1 barely beats PC2: nothing worth compressing.
